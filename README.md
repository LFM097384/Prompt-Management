# Prompt Management v0.2.1

一个现代化的提示词(Prompt)管理系统。

## 功能特点

- 支持提示词的新建、编辑、删除
- 内置提示词与自定义提示词分类管理
- JSON格式导入导出
- 现代化深色主题UI
- SQLite本地数据存储
